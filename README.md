# daynote

The daynote project is a diary application. It's possible to add notes and photos to every day and it's possible to save anniversaries on dates.

## Development

Install dependencies:

```bash
npm install
```

Start the Vite dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
